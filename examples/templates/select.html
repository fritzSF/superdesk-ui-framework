Select

Line select shares the same basic structure as the Line input. An additional modifier class sd-line-input--is-select on the sd-line-input element is required to add the caret. The select element must have the class sd-line-input__select.

Example Markup

// Select with hint text

Dolor in hendrerit.

// Select with error message

This field is required.

// Disabled select

This field is disabled.

// Alternative styling - boxed select

As with the input fields there is a boxed style for select fields as well. Same rule appies as well, append .sd-line-input--boxed to the default classes.
