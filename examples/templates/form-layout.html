Form layout

Creating a form in Superdesk is extremely easy. The basisc element for form layouts is the form__row class placed on a div. In its default state it just provides a container for a form element, with a bottom margin. The form__row container provides vertical flow and behaviour to containing elements. The modifier class form__row--flex will provide a flex-box behaviour to the container. For simpler layouts multiple sd-line-input elements can be placed directly inside the form row. For more complex layouts elements should be placed in additional form__row-item container first. The elements will take space of the parent form__row form__row--flex container. An an additional modifier form__row-item--no-grow can be added in order to mace the item take only the space of the content.

Example Markup

// Form row with a single element

Sed a nunc non lectus fringilla suscipit.

// Form row with multiple elements and the flex modifier

Lectus fringilla suscipit.
Submit

// Form row with multiple elements inside form__row-item containers

Duis autem vel dolor.
Eat your vitamins!
No veggies, no dessert!
